### 🚀 Как Начать

**Способ 1: Откройте START.html в браузере** ← САМЫЙ ПРОСТОЙ!
- Содержит инструкции и кнопки для запуска
- Работает без установок

**Способ 2: Откройте index.html в браузере**
- Прямой запуск игры
- Может потребоваться локальный сервер

**Способ 3: Используйте локальный сервер (РЕКОМЕНДУЕТСЯ)**
```bash
# Python
python -m http.server 8000

# Node.js
npx http-server
```
Затем откройте: http://localhost:8000
