# 🚀 Инструкции по запуску игры

## Быстрый старт (Простейший способ)

### Способ 1: Просто откройте файл в браузере
```
Дважды нажмите на файл index.html
```
✅ Работает сразу, без установок!

---

## Локальный сервер (Рекомендуется)

### Способ 2: Через Python (уже установлен в Windows)

**Для Python 3:**
```bash
# Откройте PowerShell в папке проекта и выполните:
python -m http.server 8000

# Или используйте python3:
python3 -m http.server 8000
```

Потом откройте в браузере: **http://localhost:8000**

**Для Python 2 (если у вас старая версия):**
```bash
python -m SimpleHTTPServer 8000
```

---

### Способ 3: Через Node.js

Если у вас установлен Node.js:

```bash
# Установите http-server глобально
npm install -g http-server

# Запустите сервер
http-server
```

Откройте: **http://localhost:8080**

---

### Способ 4: Через VS Code

Установите расширение "Live Server" и нажмите "Go Live"

---

## 📋 Системные требования

- ✅ Любой современный браузер (Chrome, Firefox, Edge, Safari)
- ✅ JavaScript должен быть включен
- ✅ Для звука нужна поддержка Web Audio API

## 🎮 Как играть

1. **Откройте игру** в браузере (index.html)
2. **Нажмите "НАЧАТЬ ИГРУ"**
3. **Введите ваше имя**
4. **Купите электросамокат и телефон** в магазине
5. **Можете купить страховки** (опционально, но рекомендуется!)
6. **Бросайте кубик** и путешествуйте к острову
7. **Избегайте сложности** или пусть страховки вас спасают!
8. **Доберитесь до острова** с максимальным количеством денег — вы выиграете!

## ⚙️ Функциональность

### ✨ Особенности игры
- 🎲 Реалистичный бросок кубика
- 🤖 3 интеллектуальных бота с разными стратегиями
- 🎨 Красивая адаптивная графика (работает на мобильных)
- 🎵 Динамическая музыка (Web Audio API)
- 🎤 Озвучка на русском языке (Web Speech API)
- 📊 Детальная статистика игроков
- 📱 Полностью интерактивный интерфейс

### 🛡️ Страховки
- **Электросамоката** - защита от поломок и угонов
- **Телефона** - защита от разбитого экрана и потерь
- **Здоровья** - оплата лечения травм и болезней
- **Билета** - возврат денег при опоздании

---

## 🐛 Если что-то не работает

### Звук не воспроизводится
→ Проверьте, что звук включен в браузере

### Голос не работает
→ Убедитесь, что текст-в-речь включен в браузере на русском языке

### Страницы не загружаются правильно
→ Используйте локальный сервер (способ 2, 3 или 4 выше)

### Canvas не рисуется
→ Обновите браузер до последней версии

## 📱 На мобильном телефоне
