# 🏝️ Путь к Острову - Игра о страховке Ингосстрах

Современный интерактивный educational game про важность страховок!

## 📋 Описание

**"Путь к Острову"** — игра, которая обучает игроков о значимости страховок на примере путешествия к острову. Игроки должны обойти препятствия, принимая решения о покупке страховок Ингосстрах.

### 🎮 Игроки
- 👨‍🎓 **Личный игрок** - вы сами управляете
- 🤖 **3 бота с разными стратегиями**:
  - 👑 **Люксер** - покупает дорогое, но без страховок (часто теряет деньги)
  - 🫥 **Ноунейм** - экономит, покупает дешевле всех (очень рискует)
  - 🛡️ **Ингострик** - умный покупатель со страховками (часто выигрывает)

## 🚀 Быстрый старт

### Запуск локально

1. **Откройте файл в браузере:**
   ```bash
   index.html
   ```

2. **Используйте локальный сервер (рекомендуется):**
   ```bash
   # Python 3
   python -m http.server 8000

   # Python 2
   python -m SimpleHTTPServer 8000

   # Node.js
   npx http-server
   ```

3. Откройте в браузере: `http://localhost:8000`

## 🎯 Правила игры

### Начало
- Начальный баланс: **100 000 💰**
- **Обязательные покупки**: электросамокат + телефон
- **Опциональные страховки**: 4 вида страховок

### Механика
- 🎲 Бросьте кубик (1-6)
- ➡️ Перейдите на клетку
- 🎯 Столкнитесь с событием (потери денег или страховка спасает вас!)
- 🏝️ Дойдите до острова (клетка 48) с максимумом денег

### События на доске
- 💔 Травмы и болезни (~3-10k 💰)
- 🛴 Поломки транспорта (~5k 💰)
- 📱 Потеря телефона (потеря цены телефона)
- 🚗 ДТП (15k 💰 + возможная травма)
- 🚢 Опоздание на корабль (нужен новый билет за 10k 💰)
- ⚡ Мелкие расходы (100-300 💰)

### Страховки
| Страховка | Цена | Покрытие | Условие |
|-----------|------|---------|---------|
| 🛴 Электросамоката | 5% | Ремонт/угон | Поломка, угон |
| 📱 Телефона | 5% | Ремонт/потеря | Разбитый экран, потеря |
| 🏥 Здоровья | 10k | Лечение | Травмы, болезни |
| 🚢 Билета | 500 | Новый билет | Опоздание на корабль |

## 🎨 Современные технологии

- ✅ **HTML5 Canvas** - отрисовка доски
- ✅ **CSS3** - адаптивный дизайн (Grid, Flexbox)
- ✅ **Web Audio API** - генерация музыки и звуков
- ✅ **Web Speech API** - озвучка на русском языке
- ✅ **Модульный JS** - разделение на логические части
- ✅ **Responsive Design** - работает на мобильных и ПК
